# MT Digital — Dijital Pazarlama Ajansı Web Sitesi

Afyonkarahisar merkezli **MT Digital** dijital pazarlama ajansının tek sayfalık tanıtım web sitesi.
Saf **HTML + CSS + JavaScript** ile geliştirilmiştir; herhangi bir derleme adımı veya bağımlılık gerektirmez.

## Özellikler

- 🎬 Afyon değerlerini (Kale, Kocatepe, Ulu Camii, Zafer Anıtı) gösteren sinematik giriş slayt gösterisi
- 🧩 Hizmetler bölümü (gerçek marka logolarıyla: Trendyol, Hepsiburada, Meta, Google Ads, Photoshop, Figma vb.)
- 🔄 "Nasıl Çalışıyoruz" süreç bölümü
- 💬 Sık Sorulan Sorular (açılır-kapanır)
- 📦 Paketler (fiyatsız, "bilgi alın" odaklı)
- 📱 Tam mobil uyumlu, WhatsApp ve telefon entegrasyonu
- ⬆️ Yukarı çık butonu, sabit WhatsApp butonu

## Kurulum

Sunucuya veya herhangi bir statik hosting'e dosyaları yüklemeniz yeterlidir.
Yerelde denemek için `index.html` dosyasını tarayıcıda açın.
